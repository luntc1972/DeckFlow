# Card Lookup

The Card Lookup page (`/card-lookup`) has two modes:

- **Single Card** (default; the only mode visible on mobile) — type a card name, get live Scryfall suggestions once you've entered 4+ characters, and picking a suggestion (or pressing Look Up) renders that card's Oracle text plus WOTC rulings inline.
- **Card List** (desktop-only) — paste up to 100 card names and download the full Scryfall output as `.txt` or structured `.json`. The inline line editor with per-row autocomplete is still available for editing before downloading.

The Single Card result panel includes an "Ask a rules question about this card →" link that deep-links into the Ask a Judge page with the card pre-selected.
